### Next Level Week

## Sobre
Projeto criado durante a Trilha Ignite da Next Level Week eSports da RocketSeat.

## Tecnologias utilizadas
- Front-end: TypeScript, React.js, Tailwind CSS
- Back-end: Node.js, Express, SQLite e Prisma ORM
